Cosas que hacer y pensar?

El proyecto podría pasarse a un generador de código o podría pasarse a un servidor. Se requiere muy poco cálculo para producir el codigo. Y como vimos, se requirió muy poco esfuerzo para escribir la programación que construyó el sistema. Podríamos hacer que la función tomara un código de operación que ayudaría en la generación de código. También podríamos hacer que tomara métodos adicionales que se usarían para hacer el plegamiento constante y la generación de código. Podríamos agregar instrucciones adicionales (en un futuro ), etiquetas de instrucción, más comprobación de errores, recuperación de errores y muchos más operadores. Podríamos agregar la especificación de tipos y la inferencia. y otras cuestiones a tomar en cuenta. Podríamos hacer que nuestro lenguaje sea extensible. Con la misma facilidad con la que podemos definir nuevas variables, podemos dejar que el programador agregue nuevos operadores y nuevas instrucciones.
